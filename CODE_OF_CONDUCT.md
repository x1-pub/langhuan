# 行为准则

## 我们的承诺

作为社区成员、贡献者与维护者，我们承诺让每个人都能在一个充满尊重、包容、无骚扰的环境中参与项目。

## 我们的标准

积极的行为示例：

- 以尊重与善意对待他人
- 接受不同观点与经验，乐于倾听
- 以建设性方式提供与接受反馈
- 关注社区整体利益，而非个人得失
- 主动帮助新人，营造协作氛围

不可接受的行为包括但不限于：

- 使用带有性暗示的语言或不受欢迎的挑逗
- 辱骂、歧视、恐吓或人身攻击
- 公开或私下的骚扰
- 未经同意披露他人的隐私信息
- 其他可被合理认定为不恰当的行为

## 适用范围

本行为准则适用于项目仓库、Issue、Pull Request、代码评审、线上线下活动，以及任何代表项目的公开场合。

## 反馈与执行

若你遭遇或目击违反准则的行为，请立即通过 `x1_mailer@163.com` 私下联系我们。维护者将及时调查并采取相应措施，保护举报者隐私。

维护者可能采取的处理步骤：

1. **口头或书面提醒**：指出问题并要求停止不当行为。
2. **正式警告**：明确说明后果，若再犯将升级处罚。
3. **临时禁言/禁用**：在一定时间内禁止参与社区活动。
4. **永久移除**：在严重或屡教不改时，永久取消参与资格。

## 署名

本准则参考 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版本，并根据项目需求进行了本地化调整。
