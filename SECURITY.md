# 安全策略

我们非常重视用户与贡献者的安全。如果你在 Langhuan 中发现安全漏洞，请遵循以下流程：

1. **不要** 在公共 Issue、讨论或 PR 中披露漏洞细节。
2. 通过邮件 `x1_mailer@163.com` 私下报告问题，并提供：
   - 受影响的模块或文件
   - 复现步骤
   - 可能的影响范围
3. 我们将在 48 小时内确认收到，7 个工作日内反馈修复计划。
4. 修复完成且发布补丁版本后，我们会感谢你的贡献，并在得到许可的情况下公开致谢。

## 支持版本

仅维护 `main` 分支以及最新的发布版本。对于旧版本，我们可能建议直接升级。

感谢你帮助我们打造更安全的生态！
